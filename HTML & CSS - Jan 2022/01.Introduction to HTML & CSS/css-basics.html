Различават се по тяхната ШИРОЧИНА и как я определяме

BLOCK - width МОЖЕ да се променя!

BLOCK - СЕДИ САМ НА РЕДА!

BLOCK - ВИНАГИ Е ОТ КРАЙ ДО КРАЙ!

BLOCK - width 550px, макар да е по тесен от цялата страница, НО ВИНАГИ СЕДИ САМ НА РЕДА!

BLOCK - ВИНАГИ СИ СЕДЯТ НА МЯСТОТО!

INLINE - width НЕ може да се променя! INLINE - ОПРЕДЕЛЯ СЕ ОТ НЕГОВОГО СЪДЪРЖАНИЕ 1232323 INLINE - width 200px, дори сменяйки му широчината, той СЕ ОПРЕДЕЛЯ ОТ НЕГОВОГО СЪДЪРЖАНИЕ INLINE - НАВИРА СЕ МЕЖДУ ДУПКИТЕ
INLINE-BLOCK - width МОЖЕ да се променя, НАВИРА СЕ МЕЖДУ ДУПКИТЕ!
INLINE-BLOCK - display: inline-block
INLINE-BLOCK - width: 700px;
INLINE-BLOCK - width: 100px;
INLINE-BLOCK - комбинация между BLOCK и INLINE
