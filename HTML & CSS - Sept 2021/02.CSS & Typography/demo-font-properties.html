The CSS font proporties styles the font of the text:

  • font-family / font-face (e.g. sans-serif)
  • font-size / line-height(височина на шрифта) (e.g. 18pt)
  • font-weight (e.g. bold) - дебелина
  • font-style (e.g. italic) - стил
  • font-variant (e.g. SmallCaps)

font-family - name

Arial ---- p.ar {font-family: Arial, Helvetica;}

Times ---- p.tim {font-family: 'Times New Roman', Times;}

@font-face - external font

Merriweather ---- @font-face {font-family: 'Merriweather'; src: url('/files/Merriweather/Merriweather-Bold.ttf');}

Generic name

A broad class of similar fonts used in a prioritized list of fonts

  • serif - all characters have stroke endings - по default
  • sans-serif - no character has stroke endings - предимно се използва
  • monospace - all characters have the same width
  • cursive - handwritten fonts
  • fantasy - decorative fonts

Font-size - defines the text size

на 49min от видео лекцията, много добре обяснено

  • px / pt values (e.g. 16px / 12pt) - 1px == 0.75pt == 1/96inch - фиксирана стойност
  • em values - relative to the original size to nearest parent, multiplied by a SCALE factor - относителна стойност до най-близкия parent
    • Полезно при - писане на статии, постове, артикали за запазване на пропорциите
  • rem values - relative to the HTML root size (the element) - относителна стойност до root "< html >" елемента
    • Полезно при - responsive design
